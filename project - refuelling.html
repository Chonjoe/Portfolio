Refueling Operations Dashboard

About This Project

This is an enterprise refueling dashboard tracking 28M liters across 36K sites, with RTO-level variance analysis and analyst performance metrics. Built with Power BI to optimize fuel distribution and identify operational bottlenecks.

What I Used

  • Power BI Desktop for data modeling and visualizations
  • DAX for variance calculations and performance metrics
  • Power Query for cleaning refueling data from 36K sites
  • HTML embed code to display on this portfolio

What I Learned

  • Volume hides variance: 28M liters total sounds great, but RTO breakdown (4.2M down to 1.6M) revealed massive performance gaps between teams.
  • Analyst tracking drives accountability: Connecting refuel counts to individual analysts helped leadership identify top performers and target coaching where it was needed.
  • RMS vs Flowmeter mismatch is real: Comparing system data to flowmeter readings exposed calibration issues across 36K sites — invisible without side-by-side comparison.
  • Historical data unlocks trends: Year filters back to 1900 enabled seasonal pattern analysis and long-term benchmarking against past operations.
  • Multi-dimensional filtering: Region + RTO + Analyst + Entity + time period — users could slice data 7 ways and find answers in seconds instead of hours.
