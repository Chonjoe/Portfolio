Diesel Monitoring Dashboard

About This Project

This is a Real-time dashboard tracking 444K liters of diesel supply, 408 confirmed Digital Waybills, and ₦19M recovered from theft. Built with Power BI to help operations teams reduce fuel loss and optimize logistics across the Zone.

What I Used

  • Power BI Desktop for data modeling and visualizations
  • DAX for calculated measures (YoY growth, running totals)
  • Power Query for data extraction and transformation
  • HTML embed code to display on this portfolio

What I Learned

  • Data quality matters: 181 sites had disconnected fuel sensors — dashboards reveal infrastructure gaps, not just metrics.
  • Context is everything: 444K supply means 7.2% of monthly allocation. Percentages make raw numbers meaningful.
  • Theft is measurable: ₦19M recovered at 60.9% recovery rate — operations teams used this to prioritize investigations.
  • DAX for time intelligence: Calculated MoM allocation changes (20.18%) and outage MTTR (01:05:23) with custom measures.
  • Dashboards beat spreadsheets: Migrated outage tracking from manual Excel to real-time Power BI — reduced reporting time by 70%.
